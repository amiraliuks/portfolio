## About
A modern personal portfolio built with Next.js 16, featuring blog posts, personal certifications, and a curated showcase of projects I’ve built and contributed to. It serves as both a professional profile and a living archive of my work, experiments, and ongoing development.

---

## Tech Stack
### Core
- **Next.js 16**
- **React 19**
- **TypeScript**
- **Tailwind CSS 4**

### UI & Styling
- **Radix UI** (Dialog, Tooltip, Avatar, Collapsible, Label, Slot)
- **Tailwind Merge**
- **tw-animate-css**
- **Lucide React**
- **React Icons**
- **Framer Motion**

### Content & Blog
- **MDX (next-mdx-remote)**
- **Sugar High** (syntax highlighting)

### Forms & Validation
- **React Hook Form**
- **Zod**
- **@hookform/resolvers**

### UX & Effects
